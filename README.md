# AMMDS

AMMDS（Adult Movie MetaData Scraper）是一款私人专属数据管理平台，专注于帮助用户轻松管理和整理个人影视库。通过智能刮削元数据、演员识别、多源数据匹配等功能，打造专属的家庭影院体验。

> [!IMPORTANT]
> 该软件为**离线产品**，数据来源由使用者**手动录入**。
>
> 请勿在国内宣传该产品，如有需要请使用简称：`AMMDS`

> [!WARNING]
> 集成插件需要**网络功能**。
>
> 如开启任一集成插件，将视为您**已同意接入网络连接**。

## 🎬 主要特性

### 🎬 影视库管理
- 支持多种视频格式，自动刮削元数据，打造个性化影视库。

### 👤 自动演员识别
- 自动识别影片中的演员信息，提供详细的演员资料和作品列表。

### 📄 自动元数据刮削
- 自动从互联网获取影片元数据，包括封面、简介、评分等。

### 📤 分享功能
- 将影视库分享给好友或家人，轻松共享你的影视收藏。

### 🔌 插件集成
- 提供丰富的插件支持，轻松集成到现有的媒体服务器或播放器中。

### 🌐 多源刮削
- 支持从多个数据源获取影片信息，确保数据全面准确。

### 🤖 智能匹配
- 通过智能算法自动匹配本地文件与在线元数据，减少手动操作。

### ⚡ 批量处理
- 支持批量处理多个视频文件，提高管理效率。

### 🏷️ 自定义标签
- 支持自定义标签，实现个性化分类与管理影片资源。
